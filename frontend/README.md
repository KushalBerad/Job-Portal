# Job Portal – MERN Stack Recruitment Platform

A full-stack job portal platform built using the MERN stack that allows users to browse jobs, apply for positions, and manage recruiter listings.

---

## Features

- User Authentication (JWT)
- Recruiter Dashboard
- Job Search System
- Role Based Access Control
- MongoDB Database Integration
- Protected Routes
- Job Posting and Management
- Responsive Frontend Design

---

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Redux Toolkit
- Axios

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Mongoose ODM

---

## Architecture

Client → API Layer → Authentication Middleware → Controllers → Services → MongoDB

---

## Security Features

- Password Hashing using bcrypt
- JWT Token Authentication
- Protected API Routes
- Environment Variable Isolation
- MongoDB Secure Cloud Storage
